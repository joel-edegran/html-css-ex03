# Exercise 3: "The Digital Parking Garage" (a11y)

This project focuses on **web accessibility (a11y)** and **search engine optimization (seo)** by refactoring a legacy codebase into semantic HTML5 and utilizing ARIA attributes.

## Topics & Tags
`a11y` `accessibility` `html5` `css3` `autoprefixer` `wcag` `seo`

## Project Purpose
The goal is to optimize a live-map interface for a digital parking garage, ensuring it is fully keyboard-navigable and machine-readable, achieving a 100% score in Google Lighthouse audits.

## Core Technologies
* Semantic HTML5
* CSS Grid / Flexbox
* PostCSS & Autoprefixer
* ARIA Attributes (aria-label, aria-hidden, aria-live)
